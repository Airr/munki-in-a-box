Munki in a Box

Post Munki Install Simple Deployment Script

The goal of this script is to deploy a basic munki repo in a simple script based on a set of common variables. I have placed defaults in these variables, but they are easily overridden and you should decide where they go.

This script is based upon the Demonstration Setup Guide for Munki, Getting Started with AutoPKG, and other sources. My sincerest thanks to Greg Neagle, Tim Sutton, Allister Banks, Rich Trouton, Charles Edge, Hannes Juutilainen, Sean Kaiser, Peter Bukowinski and numerous others who have helped me assemble this script.

Pre-Requisites:

1) 10.8/Server 2, 10.9/Server 3 or 10.10/Server 4
2) Web Services Enabled, PHP activated
3) Munki Tools Installed

Installed Packages

1) AutoPKG
2) munkireport-php
3) If munkitools is not installed, this will install core and admin modules of Munki 2
4) Sean Kaiser's AutoAutoPKG Scripts
5) munki-enroll
6) Hannes Juutilainen's MunkiAdmin.app

Directions for Use:

1) Download Script
2) Alter Lines 20-21 to reflect your choice for munki repo name and location
3) Alter Line 32 to reflect your choice of AutoPKG installs
4) Alter Line 35 to reflect your admin username (ladmin is default)
5) Alter Lines 37-38 to reflect AutoPKG Automation Scripts
6) sudo ./munkiinabox.sh

Future Features:

1) Allow check of plain apachectl for purposes of using on a non-Server.app platform.
